Quiz Application

API Endpoints

  • /api/quizzes - List all quizzes
  • /api/quizzes/teachers - List all teachers

Admin Tools

  • H2 Database Console - Manage database (JDBC URL: jdbc:h2:mem:testdb)
  • Health Check - Check if service is running

About

This is a RESTful API for a quiz application developed as part of a school project. It provides endpoints for managing quizzes, questions, choices, teachers, and more.
